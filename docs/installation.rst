Installation
============

Requirements
------------

JAX-HDC requires Python 3.8+ and the following dependencies:

* JAX
* NumPy
* (Additional dependencies will be listed here)

Installing from PyPI
--------------------

.. code-block:: bash

   pip install jax-hdc

Installing from Source
----------------------

.. code-block:: bash

   git clone https://github.com/yourusername/jax-hdc.git
   cd jax-hdc
   pip install -e .
