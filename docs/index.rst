JAX-HDC Documentation
=====================

Welcome to JAX-HDC's documentation!

JAX-HDC is a Python library for Hyperdimensional Computing (HDC) built on top of JAX,
providing efficient implementations of HDC algorithms with automatic differentiation
and hardware acceleration support.
